HTML

  • HTML stands for Hyper Text Markup Language.
  • It is the standard markup language for creating web pages.
  • HTML consists of a series of elements.
  • These elements tell the browser how to display the content.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element allows web developers to organize content on the page based on the topics.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is a programming language used by developers to define how webpage content is presented to others.
  • CSS defines the colors, aesthetics, visual layout and organizaiton of the webpage.
  • CSS is used to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is used for tracking changes locally.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
